# bania_kb

Персональная вики о русской бане — синтез по материалам банных конференций
(коллективное парение, женская и мужская баня, этика и травмоинформированность,
техники, мастера, места). Построена по паттерну incremental-вики: знание
компилируется в связный набор markdown-страниц со ссылками `[[вики-ссылки]]`,
а не выводится заново на каждый запрос.

**Сайт:** https://serg123e.github.io/bania_kb/

## Что внутри

- `wiki/topics/` — синтез по темам (тезисы, противоречия, открытые вопросы).
- `wiki/concepts/` — понятия, техники, ритуалы.
- `wiki/entities/` — люди, места, бани, проекты, события (+ `_aliases.md` — реестр двойников).
- `wiki/sources/` — страница на каждый источник (TL;DR, ключевые тезисы, связи).

Первичные источники (полные транскрипты, yt-dlp-дампы) в публичный репозиторий
не входят — публикуется только вики.

## Сайт (SPA)

Статический клиентский SPA, раздаётся через GitHub Pages. Без бэкенда и сборочного
тулчейна на стороне Pages — всё предсобрано.

- `build_site.py` — генератор: парсит `wiki/`, разрешает `[[wikilinks]]`,
  считает обратные ссылки, пишет `data/pages.json`.
- `index.html` + `assets/` — оболочка, рендер (marked), поиск (MiniSearch),
  навигация по разделам, backlinks, тёмная/светлая тема.

Пересборка после правок вики:

```bash
python3 build_site.py        # → data/pages.json
python3 build_site.py --check  # + отчёт по orphan-ссылкам
```

Локальный просмотр:

```bash
python3 -m http.server 8000   # затем http://localhost:8000/
```
