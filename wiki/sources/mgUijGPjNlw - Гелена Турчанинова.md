# Гелена Турчанинова — прикладная анатомия для пар-мастера

Эфир [[ббк-15]] («Мастерство пар-мастера», май 2025). Гелена Турчанинова из Подольска — миолог, специалист по мануальным техникам, реабилитолог и пармастер. Соединяет работу вениками с анатомией: кожа, фасция, миофасциальные линии, лимфатическая система. Ведёт [[наташа-ведущая|Наташа]].

## TL;DR
Анатомический ликбез для пармастера: кожа как трёхслойный «фильтр» (трансдермальное усвоение, магниевый раствор), фасция как «второе тело» и миофасциальные линии [[фасциальные-линии-майерса|по Майерсу]] (как класть и пропаривать гостя по цепям), и главное — лимфатическая система как «государство», которое нельзя ломать. Жёсткое правило: лимфоток только **снизу вверх**; воздействие сверху вниз ломает клапаны и узлы (Турчанинова приравнивает забитый узел к стадиям онкологии). Отдельно — диагностика по проекционным зонам Захарьина–Геда и забота о самом мастере (сгущение жидкостей при t>45°, [[очищение-лимфы-солодкой|солодка]], питьевой режим).

## Key points
- **Кожа — самый большой орган** (1,5–2 м², 4–6% массы тела), три слоя: эпидермис → дерма (главная клетка — фибробласт, «вагончик» с питанием) → гиподерма. По цвету/структуре/заломам кожи мастер читает внутренние состояния («ушки» на бёдрах — это отёки, височный залом — зажат череп).
- **[[трансдермальное-усвоение]]:** что проходит через кожу — усваивается на 100% (магний внутрь — лишь 35%). Рецепт магниевого раствора: 250 мл кипятка + 2 пакетика сульфата магния + 1 ст. ложка соли, нанести на тело и дать высохнуть. Магний — расходник пармастера. Таблетки растворять нельзя (наполнители) — только порошок.
- **Первое парение = послойная подготовка кожи.** Термическое воздействие активирует все три слоя, фибробласт начинает доставлять питание — организм успокаивается.
- **[[фасция-как-второе-тело|Фасция]]** — соединительная плёнка вокруг каждого волокна/мышцы/кости/органа. Слипшаяся кожа «затягивает» фасцию, и фибробласт «не может проехать» → нарушается питание зоны. Рекомендует Томаса Майерса («Анатомические поезда», «Фасциальный релиз»).
- **[[фасциальные-линии-майерса|Миофасциальные линии]] и укладка гостя:** при сутулости передняя линия стянута, задняя перетянута — класть **на живот** (его «нормальное» состояние), расслаблять стянутое и активировать растянутое. Разобраны функции линий: задняя поверхностная (вертикаль), латеральная (осанка/баланс — отсюда массаж ушей и стоп), спиральная (восстановление после родов, «убрать животик»), глубинная фронтальная (стабилизация суставов и дыхание), линии рук (от пальцев к плечу), функциональные (шея/голова, речь, глотание).
- **Парение по мышечной цепи:** делать припарки на крепления мышц по ходу линии → мышца расслабляется, усиливается трение тканей, растёт активность фибробластов, восстанавливается питание.
- **Проекционные зоны Захарьина–Геда** (атлас Агулова): дискомфорт в зоне = сигнал о возможном заболевании органа. Мастер должен спрашивать о заболеваниях гостя — иначе риск (потеря сознания на полке).
- **[[экстренная-помощь-на-полке|Экстренная помощь при потере сознания]]:** не паниковать, не двигать человека — оголить стопу и тупым предметом активировать точку селезёнки → запуск большого круга кровообращения → человек приходит в себя.
- **[[лимфатическая-система-в-бане|Лимфатическая система]] — центральная тема.** Лимфоток только **снизу вверх**; сосуд без гладкой мускулатуры, движется за счёт работы мышц (сжатие/растяжение/скручивание), клапаны-«воротца» пускают лимфу только вверх. Воздействие сверху вниз = ломаем клапаны → отёк, «болотце» с ядами.
- **Лимфоузел — «лаборатория»** (10 входов, 1 выход; макрофаги и лимфоциты). Механически сломанный узел не восстанавливается; забитый узел Турчанинова приравнивает к стадиям онкологии (метастазы идут по лимфе). Отсюда запрет на удары и зажимы по узлам.
- **Плацдармы сброса лимфы** (снизу вверх): влагалище/уретра → кишечник (тысячи узлов, главный «завод») → подмышки/потовые железы → нос → миндалины/гортань/бронхи → лёгкие. «Затыкая» эти выходы (дезодорант, сосудосуживающие капли) — гоним заразу глубже (мастопатия от закрытых подмышек, лимфоозёра мозга от капель).
- **Дренаж вениками** — от периферии к центру (живот), движения только снизу вверх. Соляное выкатывание сверху вниз — нежелательно (только поверхностно по коже).
- **Голова:** лимфоузлов нет, есть «лимфатические озёра», отток самотёком вниз через шею. Прогрев шейно-воротниковой зоны ярким паром ускоряет отток из головы — Турчанинова за такой прогрев в споре «парить ли голову».
- **Безопасность зон:** подмышку — только слегка прижать горячий веник (не зажимать); подколенный узел — только лёгкий прогрев без механики; валик под колени можно (сверху, не «заколачивать»).
- **Забота о мастере:** при t>45° сгущаются кровь/лимфа/межклеточная жидкость → риск. [[очищение-лимфы-солодкой|Солодка]] (1 ст.л. на стакан, натощак) + 30 мин зарядки-тряски + абсорбент (полисорб). Питьевой режим: травы с кремнием (хвощ, берёзовый/смородиновый лист, брусника), грейпфрут после парения, водородная вода.
- **Противопоказания по лимфе:** 3 года после онкооперации — не трогать; кормящим лимфодренаж нежелателен (грудное молоко = лимфа, «уйдёт ребёнку»).

## Notable quotes

- «Эта система требует обращения с собой на "вы". Только на "вы"».
- «Можно нарисовать портрет... определить облик человека только по фасциальной сети — это полное повторение нас внутри. Это ещё одно тело».
