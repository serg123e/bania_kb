# Александр Мартынов — проектирование состояния гостя через банную программу (эффект «Ого!»)

Эфир [[ббк-10]] («Баня впечатлений», май-июнь 2024). [[александр-мартынов|Александр Мартынов]] (Санкт-Петербург, «Красная Шапка») — ex-программист → банный продюсер, вице-чемпион СПб, девиз «меньше работать, больше зарабатывать». Ведёт [[наташа-ведущая]]. **4-й эфир Мартынова** в проекте — после [[ббк-2]], [[ббк-7]], [[ббк-11]].

## TL;DR

Программистский взгляд на банную программу как **алгоритм проектирования состояния гостя**: цель (запрос) → последовательность заходов-«методов» → гарантированный результат, дающий эффект **«Вау»**. Три задачи: выделиться среди тысяч мастеров, попасть в реальный запрос гостя, запустить рекомендации. Метод старта: **прожить опыт на себе** (программа «Тишина» родилась из личного отключения головы в Грузии), разложить его на действия, переложить на баню. Каркас — **4 захода: расслабление → очищение → наполнение → баланс**, где наполнение каждого — личный опыт мастера (= его УТП). Ключевые приёмы: **не перегружать** (выкидывать действия без понятного смысла), **узкое говорящее название** = решение запроса = +вдвое аудитория, **тестировать гипотезу** на знакомых до запуска, собирать **обратную связь** как code review. Разбирает две версии мужских программ — тонизирующую и «Тихую мужскую радость».

## Key points

- **Цель эфира — спроектировать состояние гостя**, чтобы он вышел с эффектом «Ого! Лучшая баня в жизни». Своя программа = уникальное преимущество: по программе сразу «считывается» мастер, его видение и ценности.
- **Типичная банная программа** (намерение на первом пару → галотерапия/соль → парение вениками → медитация) — не плохая, но «как у всех»; от перегруза ею становится скучно, мастер устаёт.
- **Запрос гостя — отправная точка.** Источники: слушать гостей дословно (фиксировать), ритуал первого пара (частотность запросов), **отзывы** (важнейший вклад гостя), наблюдение конкурентов, подслушивание в кафе и тематические паблики.
- **Самый частый запрос Мартынова — «отключить голову».** Под него создана программа **«Тишина»** (родилась из личного опыта в Тбилиси: душ → сауна → ароматы → чай → серный источник → контраст → тишина лёжа → воздух).
- **Проектирование состояния идёт через себя:** «как понять, что чувствует другой? — понять, что чувствуем мы; большинство чувствуют так же». Состояния не скачут — к «тишине в голове» ведут постепенно (первый шаг — переключить внимание на внешние органы чувств).
- **Понимать назначение каждого действия** — как программист знает результат каждой строки кода. Действие без понятного смысла → выкинуть или проверить, отвечает ли цели.
- **Ритуалы как основа программы:** личный ритуал (прогулка по лесу, приезд к бабушке на дачу, яркое воспоминание) раскладывается на ощущения и переносится на заходы (кейсы «Прогулка по лесу», «дальняя дача с бабушкой»).
- **Одна отработанная программа лучше многих новых:** проводится всё легче, проще делегировать, мастер меньше устаёт, гости идут «за стабильным результатом».
- **4 захода = 4 компонента:** что тебя **расслабляет** → **очищает** → **наполняет** → **балансирует**. Наполнение — твой личный опыт (нутрициология, дыхание, зарядки) = УТП. Собрать кусочки и проверить, решают ли они исходную цель.
- **Не перегружать.** Стоимость оправдывает **результат**, а не «впихнуть все навыки». Кейс: убрали все индивидуальные парения из коллективной программы — гостям понравилось, цена та же, сил потрачено меньше. «Как будто выдаёшь исходный код.»
- **Перегруз ведёт к выгоранию:** тело отказывается собирать бани («головой хочу, тело — давай отдохнём») → падение дохода и стагнация.
- **«Хочу ли я сам прийти на свою баню?»** — финальный критерий. Где спотыкаешься при чтении программы — там переписать.
- **Описание программы — глаголами-действиями** (рождают образ, который гость «примеряет»); избегать отглагольных «погружение», «восстановление». Прогон образного путешествия по программе; описывать своими словами, на языке своей аудитории.
- **Название = навигация и решение запроса.** «Отключить голову» → «Тишина»; «дайте маме побыть в тишине»; «Хочу на ручки» (выстрелила в Брянске); спа-меню «Мне полегче / То что надо / Мне покрепче» или rare/medium/well done. Облегчает выбор — не манипуляция.
- **Узкий запрос собирает ~вдвое больше аудитории, чем «решаю всё».** «Я работаю с любым вопросом» отталкивает; целевой запрос (бегуны → «восстановление после соревнования») понятно, кому и где предлагать.
- **Тестировать гипотезу до запуска** на знакомых/родных (они платят аренду+расходники, дают обратную связь). Только потом — «в массы»; программа «набирает вес» после проведения и корректировки.
- **Обратная связь = code review.** Брать постоянно: «Какие мысли в голове после программы?» → «Ни одной» = результат достигнут. Гости ценят видимый рост мастера от бани к бане.
- **Делегирование через алгоритм:** зная всю последовательность, передаёшь ученикам; сейчас сам проводит 1–2 коллективных захода, сидит в зоне отдыха, парят ученики. Главное — не потерять качество (= достижение заложенного результата).
- **Две мужские программы:** (1) **тонизирующая утренняя** — 30 мин зарядки на морозе («первый жар добудь сам») → купель → пар с донником → дыхание по [[дыхание-вима-хофа|Виму Хофу]] → коллективный рык (поэтапно: гудение → массаж лица → расслабить челюсть → рык из груди) → объединяющие кличи → тишина; (2) **«Тихая мужская радость»** — медитативный релакс, прогрев сердечной зоны вениками и солью («переключение на сердечный центр»), многие плакали.
- **Универсальная формула:** любую коллективную программу можно провести и индивидуально (солью натирают всех ↔ одного). Называет каждый заход смыслом («Вдох», «Выдох», «Наполни силами», «Сила рода»).
- **Смысловая нагрузка — главный тренд.** Все делают те же действия (веник, соль, вода), но **смысл**, заложенный мастером, определяет ценность и качество внимания. «Парение сродни медитации — настолько включён в человека, что у него метаморфоза» (программа **«Метаморфоза»**, тест индийских мыльных орехов вместо дегтярного мыла).
- **Что гости отмечают в отзывах (4 пункта):** место → пармастер → состояние → результат. Выписывать эти слова → описывать баню их языком и поднимать цену.
