# Режимы парения по психофизиологическому состоянию

Выбор силы воздействия вениками по **текущему состоянию** гостя, а не только по конституции. [[иван-ивакин|Иван Ивакин]] ([[ббк-4]]) выделяет два решающих фактора и пять режимов.

## Тезисы

- **Два фактора** (из множества обследованных в [[ведапульс|ВедаПульс]]):
  - **Запас жизненных сил** (системы физиологической регуляции): 0–10000; норма 1500–3000; >5000 — застой энергии.
  - **Уровень стресса**: зелёная зона ≤~120 норма; >200 (жёлтая/красная) — высочайшее напряжение.
- **Пять режимов** (сила, амплитуда, частота, контрасты, перевороты, поддержки):
  - **Энерговосстанавливающий** (обессиленный): минимум всего, без ударных техник и контрастов, один переворот, поддержки рук/ног, образы наполнения силой.
  - **Энергобалансирующий** · **классический** (средний) · **активный** · **интенсивный** (для застоя энергии: резкие сильные контрасты, «выбивать дурь», несколько переворотов, без поддержек).
- Логика: сильное махание = стихия воздуха → перевозбуждает вата-дошу; контрасты «забирают силы» — обессиленному противопоказаны, человеку в ресурсе/застое нужны.
- Конституция ([[аюрведические-типы-конституции]]) задаёт **влажность/температуру**, состояние — **режим воздействия**. Вместе → оптимальный лёгкий пар.
