# Банная тусовка (комьюнити вокруг бани)

Объединение людей по теме бани/общего ритуала, которое мастер сознательно строит вокруг себя или своего комплекса. Для пармастера — **инструмент**, закрывающий три базовые задачи: привлечение клиентов, лёгкая продажа, [[возвращаемость-через-комьюнити|возвращаемость]]. Снимает тревогу нестабильного дохода («то пусто, то густо»).

## Ключевое (по [[александр-мартынов]], [[3ovXdpnIGWQ - Александр Мартынов]])

- **Цель первична** — прежде чем строить: объединить людей / обмен опытом / совместные вылазки / решение задач участников. Может родиться постепенно.
- **Форматы:** бесплатные, платные, условно платные (благотворительность от прибыли — клуб «Патриоты», Красная Поляна). По аудитории: мужские, женские, семейные, межгендерные.
- **Основа = [[ритуал-как-точка-притяжения|личный ритуал]] мастера**, а не абстрактный план.
- **Регулярность** (один день/время в неделю) превращает тусовку в часть жизни участников и даёт прогнозируемый доход.
- **Дополнительная ценность вне бани:** общение, контакты, рост гостя.
- Частный случай для коллег — [[клуб-пармастеров]].

## Кейсы

- «Хорошие люди» (СПб): из 4 друзей до 600–700 человек на общем ритуале (лес, прорубь, [[дыхание-вима-хофа]], завтрак).
- Мужская баня Мартынова: 3 года, раз в неделю, выросла в направление поддержки мужского здоровья.
