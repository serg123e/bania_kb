# Аюрведические типы конституции

Принцип индивидуального подхода к телу гостя по преобладающему типу нервной регуляции/конституции. У [[дмитрий-добровольский|Дмитрия Добровольского]] ([[ббк-10]]) — основа для выбора стратегии и тактики парения.

## Тезисы

- Люди различаются по доминанте: **ЦНС / симпатическая / парасимпатическая** нервная система — каждого парят по-разному, у каждого свои «проблемы».
- Образно — «Человек Огня / Ветра» и т.п.; внешний тип телосложения (астеник/гиперстеник) может не совпадать с внутренним.
- Тип определяет выбор веника (берёза — для мышц «человека огня», липа/мягкие — для астеника и ЦНС), бани (сухая/влажная), темпа наращивания нагрузки.
- Уточняется через [[нумерология-в-бане|банную нумерологию]] (по дате рождения) — Добровольский связал нумерологию с типом конституции, чтобы снять диссонанс «внешность ≠ реакция».
