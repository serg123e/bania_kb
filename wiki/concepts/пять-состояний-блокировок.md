# Пять состояний-блокировок

Пять эмоциональных состояний, которые блокируют отношения (муж↔жена, родители↔дети, между людьми) — по [[иван-ивакин|Ивану Ивакину]] ([[ббк-3]]). У каждого своя телесная/энергетическая локализация, что задаёт адресную работу в бане.

## Состояния и локализация

| Состояние | Суть | Локализация |
|---|---|---|
| **Страх** | страх смерти; сковывает, мышечный спазм | нижний центр |
| **Стыд** | себя перед собой («студить», холод, лишение жизни) | низ живота |
| **Вина** | перед другими; делает управляемым/безвольным | живот |
| **Обида** | нарушена справедливость («дева боли») | сердечный узел |
| **Ненависть** | «не» + «видеть» — не хочешь видеть обидчика | грудь/горло |

- Обида копится с детства (песочница), тащит «инструменты мести». Обида и ненависть — «две подруги».

## Работа в бане

- **Страх / обида / ненависть** = энергетический застой → ударные техники, активные действия; яркий пар по конституции.
- **Стыд** (холод) → температура + влажность («растопить»).
- **Вина** → прогрев живота/крестца/талии, припарки, круговые движения.
- Перед баней — проговорить ([[световид]], душевная беседа).
