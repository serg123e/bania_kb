# Уровни контактного воздействия вениками

Три уровня глубины контактного парения по [[ида-дмитриева|Иде Дмитриевой]] — не про «интенсивность», а про глубину захвата ткани.

## Уровни

1. **Периферия** — работа только с кожей: парение листиками по поверхности тела, лёгкое взаимодействие, почти без мышц.
2. **Средний контакт** — вибрация в мышечную ткань.
3. **Глубокий контакт** — работа с **фасцией** (держит мышцу в эмоциональном «доступе»): мышца «двигается с глубиной», растирка с широким захватом и большой вибрацией. Фасция несёт функцию восстановления → «оживляет» зону.

## Очерёдность (принцип безопасности)

- Начинать со **среднего** контакта: сбор целостности, важные точки опоры — стопы, поясница, межлопаточная зона.
- **Лёгкое периферийное парение — только после опоры**: иначе «отлётные» ощущения (печаль, грусть) всплывают без ресурса и гость проваливается.
- Глубокий контакт можно давать с контрастом; разные эффекты комбинируются в один заход или разносятся по заходам.
