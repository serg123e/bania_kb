# Энергобезопасность в бане

Тема [[круглый-стол-энергобезопасность|круглого стола]] [[ббк-3]]: как мастеру «обрядово-целительской» школы сохранять себя и не навредить гостю при работе с «потоками», «сущностями», эмоциональными выходами.

## Сквозные тезисы участников

- **Сначала себя** — [[ресурсное-состояние-мастера|ресурс]] / [[триединство-я-в-духе|«я в духе»]] / свой поток ([[света-скалозубова|Скалозубова]], [[елена-шорох|Шорох]]).
- **Без [[границы-запроса-на-исцеление|запроса]] не исправлять** — будут [[откаты-после-исцеления|откаты]].
- **Неожиданный поток** — свернуть, «не сейчас».
- **Базовая техника** — [[работа-промежностью]] (Шорох).
- **Инструменты самозащиты**: включение/удержание потока, [[соль-как-инструмент-мастера|работа с солью]] (Михаил), постоять в тазу с солью (Михайлова).
- **Несколько мастеров** — [[грунтование-общего-пространства|каждый занимает всё пространство]], не делит.
- **Духи места** — приходить «широким, открытым, с подношениями, на равных»; ключ — [[юмор-как-инструмент|юмор]].
- **Физиология за «энергией»**: застои энергии = застои крови/лимфы, запуск через тепло-холод; «баня не для нагрева — для охлаждения».
