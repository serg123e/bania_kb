# Упаковка пар-мастера

Система позиционирования банного мастера — целостный образ, по которому гость, ещё не приходя, решает обратиться. Разбор [[александр-мартынов|Александра Мартынова]] («Красная Шапка») в завершающем эфире [[ббк-3]]. **Самая ранняя** точка его бизнес-линии в проекте.

## Три опоры

1. **Что ты за мастер** — услуги и **результат** (а не «приёмы»/«лимфа течёт»), личное отношение к бане.
2. **Кто твои гости** — изучить запросы (список вопросов до/после бани), **сузить нишу** → выше чек и доверие.
3. **Уникальный атрибут** — своя «фишка» (красная кепка, звук/пение, шоу-баня, блюдо), по которой запоминают.

## Принципы

- **Главное — продукт**; упаковка без понимания «что даю» не сработает.
- **«Душевность может стоить дорого»** — занижение цены ведёт к [[выгорание-пармастера|выгоранию]] (мотив «художник не должен быть голодным»).
- Упаковка ≠ только соцсети: одежда, стол, парная, коммуникация, правила поведения («договориться на берегу»).
- **Одна соцсеть** до первых 100–200К ₽; писать **глаголами** и в настоящем времени («ты уже прогреваешься»). Книга «Пиши, сокращай».
- Площадки: YouTube, Teletype (превью сайта), Instagram (шапка по ключам + гео), сайт (нужен трафик), Telegram (навигация, читаемые посты).
- Сарафанное радио и несколько «точек касания» для высокого чека. Гастрономия/иные навыки как точка входа (кейс Виктории, СПб).
