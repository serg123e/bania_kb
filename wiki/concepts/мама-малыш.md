# Баня «мама + малыш»

Формат бани для кормящей мамы с младенцем/маленьким ребёнком ([[наталья-михайлова]], [[ббк-3]]). Продолжение работы с беременными — «они оказываются с малышами».

## Принципы

- **«Припуски на швы»:** план рушат дети. Один заход мамы = успех; 3–5 минут одной — «вау/победа». «Концентрация женской стихии вокруг мамы с малышом — максимальная… эпицентр стихии».
- **Снятие тревожности (не до нуля):** часто не закрывать дверь (мама всё равно слушает ребёнка); разрешить прервать парение и выйти на полудвижении — снимает половину тревожности.
- **Помощник** (няня/бабушка/муж) для подкорма у груди — идеально.
- **Безопасное пространство:** свои одеяла (срыгивания), убрать всё опасное/несъедобное, «баня должна быть разрешённой»; неустранима только **печка** (обезопасить лавкой/стульями).
- **Веник ребёнку** — отдельный заранее; дети тянутся к «рабочим заряженным» вещам (настоящая кочерга).
- **Аллергии/угощения** уточнять заранее; чем занять (веточки в 4–5 лет, телефон в крайнем случае — «не воспитывать мамочку»).
- **Приоритет:** задача — мама → малыша можно не заносить; задача — парить детей → мама по остаточному принципу.
- Возраст/температура: 60°C — нет, 40–45°C — да, при наличии смысла (правило 50/50).
