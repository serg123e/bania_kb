# Невербальная коммуникация мастера

Осознанное управление языком тела мастера и считывание тела гостя. Разбор [[валерия-орлова|Валерии Орловой]] (хореограф) в [[ббк-7]], адаптированный по этапам банной программы. Базовая пропорция: **55% информации — через тело, 38% — интонация, 7% — слова**.

## Тезисы

- **Поза ↔ состояние двусторонне:** приняв «уверенную» позу (жест «шпиль»), можно поймать нужное состояние перед баней.
- **Первые 4 минуты** создают впечатление почти на всё общение. Улыбка (искренняя — с мышцами глаз), открытые позы, ладони видны.
- **Личное пространство** (зоны: интимная 15-45 см, личная 46-122, социальная 122-360). На встрече — рукопожатие, не объятия (вторжение в интимную зону → адреналин, оборона). Объятия — на проводах.
- **Стол:** угловая посадка ~45° (контакт + лёгкий барьер), не напротив (противоборство); не сидеть спиной к открытой двери/окну (тревога с «пещерных времён»).
- **Чтение состояния = насмотренность**, не «видение»: мозг считывает позу как «красный цвет». Интонация одного слова «хорошо» различает кайф / нехватку / «пора завершать».
- **Коррекция закрытой позы физически:** тазик с горячей водой / подставка для ног раскрывают гостя.
- **Вход в интимную зону при парении** — мягко, через воду/веники/полотенце; проговаривать «освобожу крестец», не «сниму трусы».
- При **искреннем интересе** к гостю невербалика выстраивается сама — следить специально нужно только когда устал.
