# Дозирование впечатлений

Принцип построения спа-программы по [[татьяна-шиянова|Шияновой]] ([[ббк-15]]), взятый из **художественной композиции**: первый/второй/третий план — большое, среднее, малое. «Когда всё одинаково ярко — глазу не за что зацепиться, и ему становится скучно». Нельзя «делать всё одинаково хорошо» — нужно чередовать яркое и тихое, чтобы гость не перегрузился и захотел «задержаться и погрузиться».

Перегруз сенсорной системы — тоже рабочий инструмент, но только для «супернапряжённых» людей (перегрузить, чтобы перезагрузилась); другим нужно просто **напитать** (в пределе 2–3 ощущений). Связано с [[проектирование-состояния-гостя]], [[действие-и-бездействие]], [[пять-чувств-в-бане]].
