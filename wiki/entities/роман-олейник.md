# Роман Олейник

Профессиональный пар-мастер из Санкт-Петербурга, специалист по коллективному парению. Идейный вдохновитель проекта **Шоу-баня**, режиссёр-постановщик и ведущий актёр «иммерсивной бани впечатлений».

## Подход

- Баня как **[[иммерсивная-баня|иммерсивный театр]]**: всё подчинено [[сюжет-в-парении|сюжету]] (вода, лёд, еда, музыка, реквизит).
- Полимодальное воздействие на все пять чувств; гость — соучастник истории.
- Использует **[[нейросети-как-инструмент-сценариста|нейросети (Perplexity AI)]]** для сочинения сценариев.
- Маркер успеха — аплодисменты в финале; продвижение через кастомные жетоны с QR-кодом.
