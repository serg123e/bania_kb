# Гелена Турчанинова

Миолог, специалист по мануальным техникам, реабилитолог и пармастер из Подольска. Соединяет парение с анатомией: кожа, [[фасция-как-второе-тело|фасция]], [[фасциальные-линии-майерса|миофасциальные линии Майерса]], [[лимфатическая-система-в-бане|лимфатическая система]]. Преподаёт прикладную анатомию для пар-мастеров.

## Позиция

- **Лимфа — только снизу вверх.** Воздействие сверху вниз ломает клапаны/узлы; забитый узел приравнивает к стадиям онкологии. См. [[лимфатическая-система-в-бане]].
- **Кожа как фильтр** — [[трансдермальное-усвоение]] (магниевый раствор усваивается на 100%).
- **Фасция — «ещё одно тело»**; укладка и парение гостя по [[фасциальные-линии-майерса|мышечным цепям]].
- Диагностика по [[проекционные-зоны-захарьина-геда|проекционным зонам]]; [[экстренная-помощь-на-полке|реанимация через точку селезёнки]].
- Самогигиена: [[очищение-лимфы-солодкой|солодка]], питьевой режим (травы с кремнием, грейпфрут).
