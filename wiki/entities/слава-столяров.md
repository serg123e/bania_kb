# Слава Столяров

Пар-мастер из **Санкт-Петербурга**, банное пространство **«К себе»** (работает с женой Натальей). Бывший логист/предприниматель (курьерская служба) — за 5 лет стал банщиком, «который знает о бане всё». Специализация — **психосоматика** в бане. Обучался **генеративному райх-коучингу** у наставника Паши; учится у лекаря. Спикер [[ббк-7]] (тема «Психосоматика: работа с зонами мышечного напряжения»).

> В транскрипте — «Слава». Полное имя — вероятно Вячеслав; канон страницы оставлен `слава-столяров` по имени файла источника.

## Подход

- **[[психосоматика-в-бане]]** на базе теории Райха: **[[мышечный-панцирь]]**, **[[семь-мышечных-поясов]]**.
- Триада **информация → энергия → вещество**; зажим = непрожитая эмоция, застрявшая в мышце.
- **[[райх-коучинг]]** (3–5 ч, отдельный сеанс, не в бане) для собственников/руководителей.
- **[[бесы-как-части-личности]]** (работа с тенью), **[[крылья-души]]** (два источника энергии).
- В бане — поверхностный уровень прогретыми вениками; глубину (секс/власть/деньги) — на отдельных сеансах.
- Обучает программе парения **[[экспресс-перезагрузка]]** (2 ч); опора на Проппа, сказкотерапию.
- Принцип **[[свобода-воли-в-работе]]**: работает только с готовыми.
